DATA SCRAPING TO EXCEL
USING PYTHON

Project Overview:
In this project, I undertook the task of collecting essential information about KNX partners, including Company_Name, City, Address, Person_Name, Telefon, Handy, and Website. The challenge was that the email information was not available, necessitating the extraction of email addresses from the complex nested HTML structure of the respective websites.

Project Objectives:
1. Collect Company_Name, City, Address, Person_Name, Telefon, Handy, and Website information of KNX partners and store them in an Excel sheet.
2. Develop a solution to extract email addresses from the complex nested HTML structure of KNX partner websites.

Project Implementation:
1.Data Collection:

  • Created a custom web scraper to extract data from the KNX partner's website
    (https://www.knx.org/knx-de/fuer-fachleute/community/partner/).
  • Utilized Python's BeautifulSoup and requests libraries to navigate and extract relevant information from the complex HTML structure
  • Successfully collected Company_Name, City, Address, Person_Name, Telefon, Handy, and Website details and stored them in an organized Excel sheet.

2.Email Extraction:
  • Developed a separate algorithm to extract email addresses from the KNX partner's websites.
  • Utilized regular expressions and string manipulation techniques to locate and capture email addresses from the nested HTML content.
  • Implemented error handling and validation to ensure accurate email extraction.

    • Challenges and Solutions:

    • Complex HTML Structure: The nested HTML structure of the websites posed a challenge in extracting specific data. I addressed this by carefully analyzing the DOM hierarchy and using appropriate tags and classes to navigate the content accurately.
    • Missing Email Information: Since the email information was not directly available in the KNX partner data, I had to implement a custom email extraction solution. Regular expressions and careful pattern matching helped in successfully capturing email addresses.

    Results and Outcomes:

    • Successfully collected and organized Company_Name, City, Address, Person_Name, Telefon, Handy, and Website information of numerous KNX partners in an Excel sheet.
    • Extracted email addresses from the respective websites and associated them with the corresponding partners in the Excel sheet.

    Tools and Technologies Used:

    • Python programming language
    • BeautifulSoup and requests libraries for web scraping
    • Regular expressions for email extraction
    • Selenium for Handling Browser Elements
    • Microsoft Excel for data storage and organization

    Conclusion:
    This project showcases my proficiency in web scraping, data collection, and data extraction from complex HTML structures. It also demonstrates problem-solving skills and the ability to create effective solutions for obtaining valuable information from web sources. The project's success in gathering and organizing data from KNX partners, along with the added capability of email extraction, highlights my expertise in handling real-world data challenges.
